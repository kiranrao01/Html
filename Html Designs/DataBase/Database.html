What are Database
A structure set of data

Databases are a collection of information, they are used for creating, manipulating, and deleting data. There are many different tyes of databases and they can be classified in a basic approach of relational database or a non-relational database. Relationships means a set of our data is related to another section of our data. Our databases are put into sections of data called schemes, inside each schema each data set is organized intro tables, those tables contain columns and rows to house that particular set of data.
